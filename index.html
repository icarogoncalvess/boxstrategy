O Coração do Projeto

O Box Strategy nasce para subverter a lógica de exclusão do automobilismo. Sabemos que o acesso às pistas hoje é extremamente difícil devido aos altos custos. Nossa missão é democratizar esse universo, mostrando que a tecnologia é o grande equalizador: se o asfalto é caro, o conhecimento técnico e a lógica de programação são horizontes abertos para todos os jovens de escolas públicas.
